# STM32H750屏幕一体板
## 简介
这是我在2025年3月做的一个STM32H750屏幕一体开发板，板载SDRAM，QSPI FLASH，SD卡，DAP-LINK，LCD屏幕，OV5640摄像头。板子参考反客科技设计，可运行LVGL等多种图形化界面系统。
![](./picture/cover.png)
在2024学习完嵌入式Linux和FPGA后，我一直好奇板子的硬件电路结构和硬件设计，在好奇心的和网上各位开源大佬工程的熏陶下，我开始学习高速电路设计与layout。通过观看教程 我学习了高速PCB的设计技巧，BGA芯片的封装的扇孔布线技巧，传输线和计算阻抗匹配方法，蛇形等长和差分等长的绘制技巧，电源铺铜技巧，了解了电源拓扑，DDR拓扑结构和EMC电磁兼容的大致内容，板子制作过程碰到了很多困难，USB忘记串接电阻，LCD接口画反，错把无源晶振当有源晶振焊接造成电源短路，FLASH芯片型号选错，二极管封装画反等问题接连出现，在不断调整和改进下，板子的外设和LVGL终于全部跑通了！

## 使用方法
### 屏幕一体板v1 文件夹
第1版屏幕一体板的AD工程，已打板测试，阻抗不限，测试时在捷配进行打板<br>
此版本有LCD接口画反、USB信号线忘记上拉电阻导致无法通信、FLASH 芯片封装错误等问题，不建议使用。

### 屏幕一体板v2 文件夹
第2版屏幕一体板的AD工程，已打板测试，阻抗不限，测试时在捷配进行打板<br>
此版本改进上一版的问题，在焊接的时候因为在有源晶振位置错焊无源晶振，导致板子短路。后续重装主芯片和SDRAM后开始出现SDRAM读写失败的问题。此版本未经过充分测试，不建议使用。

### 屏幕一体板v3 文件夹
第3版屏幕一体板的AD工程，已打板测试，阻抗不限，测试时在捷配进行打板<br>
此版本在上一版的基础上，独立了各个外设电路的电源域，通过0欧姆电阻相连。后续发现3.3V电源不稳，导致有概率无法和电脑进行USB通信。经过排查初步判断因为输入的5V电源的滤波电容不够引起的。
